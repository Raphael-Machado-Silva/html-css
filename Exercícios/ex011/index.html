Imagem Dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente diminua ou aumente o tamanho do seu navegador.

Para introduzir uma imagem dinãmica é necessário que se abra a tag PICTURE e dentro da mesma abra o código SOURCE MEDIA TYPE, mas antes inserir a imagem desejada dentro dessa tag PICTURE por meio da tag IMG, logo essa tag source entra como uma adaptação ao tamanho da tela e a imagem escolhida, com isso você deverá selecionar uma imagem de tamanho maior ou menor como substituta no espaço "srcset."

Lembrando a base é a foto que eu quero colocar inicialmente, seja a maior ou a menor, e acima você vai colocar os sources em ordem racional, seja a mesma, crescente ou decrescente.
