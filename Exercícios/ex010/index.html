Usando Links Externos

Você pode acessar o meu repositório público no GitHub.
Para criar um, link externo apenas é necessário envelopar a parte escolhida e colocar a tag A (de âncora) e colocar o endereço, e caso use a tag TARGET_BLANK o site vai abrir em uma nova aba (obs:essa tag deve ser usada logo após o link).

Você também pode acessar o canal do curso em vídeo pelo seu próprio site.

Usando Links Internos

Esta é a primeira pag do site, se você quiser também pode acessar a minha segunda página, apenas é necessário criar uma segunda página .html e inseri-la aqui com o código A, e para abrir essa nova página na mesma guia é neecssário após colocar a localização inserir a tag TARGET=SELF.
