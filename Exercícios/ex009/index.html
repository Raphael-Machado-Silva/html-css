Trabalhando com Listas

Listas Ordenadas

  1. Acordar;
  2. Tomar Café;
  3. Escovar os dentes;
  4. Lavar louça;
  5. Estudar;
  6. Ir tomar banho;

Para criar uma lista basta digitar a tag OL e dentro do mesmo abrir várias tags LI, para editar a formatação da lista digitar TYPE dentro da tag OL.

Listas Não Ordenadas

  • Comprar Pão;
  • Comprar Feijão;
  • Comprar Leite;
  • Comprar Camisinha;

Para criar uma lista não ordenada, apenas use a tag Ul, podendo editar igualmente o exemplo das ordenadas, usando a tag TYPE dentro do UL.

Também como visto acima, pode-se abrir uma lista dentro de outra, realizando uma outra abertura da tag OL, dentro de uma já existente.
